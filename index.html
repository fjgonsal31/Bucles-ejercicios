Tutorial: Bucles en JavaScript

Los bucles son estructuras de control que permiten ejecutar un bloque de código repetidamente. En JavaScript, tenemos varios tipos de bucles, siendo los más comunes: for, while y do-while.

1. Bucle for

El bucle for se utiliza cuando conocemos el número de iteraciones que queremos realizar.

            
            for (let i = 0; i < 5; i++) {
                console.log(i);
            }
            // Imprime: 0, 1, 2, 3, 4

El bucle for consta de tres partes:

  • Inicialización: let i = 0
  • Condición: i < 5
  • Expresión final: i++

2. Bucle while

El bucle while se utiliza cuando queremos que el código se repita mientras se cumpla una condición, sin saber necesariamente cuántas veces será.

            
                let i = 0;
                while (i < 5) {
                    console.log(i);
                    i++;
                }
                // Imprime: 0, 1, 2, 3, 4

El bucle while evalúa la condición antes de cada iteración. Si la condición es falsa desde el principio, el código dentro del bucle no se ejecutará nunca.

3. Bucle do-while

El bucle do-while es similar al while, pero garantiza que el código se ejecute al menos una vez, ya que la condición se evalúa al final de cada iteración.

            
                let i = 0;
                do {
                    console.log(i);
                    i++;
                } while (i < 5);
                // Imprime: 0, 1, 2, 3, 4

Este bucle es útil cuando queremos que el código se ejecute al menos una vez, independientemente de la condición.

Comparación de los bucles

  • for: Ideal cuando sabes cuántas veces quieres que se repita el bucle.
  • while: Útil cuando no sabes cuántas iteraciones necesitarás, pero quieres continuar mientras se cumpla una condición.
  • do-while: Similar al while, pero garantiza al menos una ejecución del código.

La elección del tipo de bucle dependerá de la situación específica y de lo que intentas lograr en tu código.
